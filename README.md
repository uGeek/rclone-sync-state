# rclone-sync-state

## Instalación
```
sudo curl -L https://raw.githubusercontent.com/uGeek/rclone-sync-state/master/rclonesyncstate \
          -o /usr/bin/rclonesyncstate && sudo chmod +x /usr/bin/rclonesyncstate
```
